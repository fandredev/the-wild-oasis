## Clone this repo

###### Notes: Check the .env.example before clone this repo

```
git clone git@github.com:fandredev/the-wild-oasis.git
```

## Go to directory

```
 cd the-wild-oasis
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```
